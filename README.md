# Claude Code GCP Template

## 1. Overview
Reusable repository template for Google Cloud, data engineering, RAG/agentic systems, web applications, governance artifacts, and secure CI/CD.

## 2. Project Type
This template is designed to support one or more of the following:
- Data Engineering
- RAG / Agentic System
- Web Application
- Data Governance
- Cloud Deployment

## 3. Core Stack
- Languages: Python,SQL,TypeScript,JavaScript,Bash,YAML,Terraform
- Frameworks: FastAPI, React, Next.js, Dataform
- Cloud services: BigQuery, GCS, Cloud Run, Pub/Sub, Secret Manager, Artifact Registry, Cloud Build, Workflows, Vertex AI
- CI/CD: GitHub Actions + Cloud Build
- IaC: Terraform
