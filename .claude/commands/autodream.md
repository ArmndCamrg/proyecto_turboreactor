# /autodream

Use this command for structured solution ideation before implementation.

## Steps
1. Frame the problem.
2. Generate 2 or 3 feasible solution patterns.
3. Compare them by:
   - speed
   - complexity
   - cost
   - maintainability
   - cloud fit
   - security
   - scalability
4. Recommend the best MVP path.
5. State what should be deferred.

## Rules
- Do not implement by default.
- Keep the recommendation concrete and practical.
