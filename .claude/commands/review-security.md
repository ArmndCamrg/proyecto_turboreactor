# /review-security

Use this command to perform a focused security review.

## Review areas
- secrets exposure
- dependency vulnerabilities
- insecure defaults
- weak input validation
- auth/authz gaps
- container hardening
- IAM over-permissioning
- public exposure in infra
- Terraform misconfigurations

## Output
- Findings
- Severity
- Affected files
- Recommended remediation
- Safe quick wins
