# /ralph-loop

Use this command for iterative implementation.

## Loop
1. Restate target.
2. Implement the smallest safe increment.
3. Validate.
4. Run relevant checks.
5. Fix issues.
6. Update docs.
7. Summarize status.
8. Recommend next increment.

## Rules
- Prefer small reversible changes.
- Do not skip validation.
- Ask for approval before high-risk commands.
